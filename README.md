# Asociación Civil Sueños

Página web institucional para la Asociación Civil Sueños.

## Descripción

Sitio de acercamiento a la asociación, pensado para que cualquier visitante pueda conocer su misión, sus actividades y novedades, y para que el equipo de la asociación pueda gestionar el contenido (noticias y actividades).

El sitio cuenta con las siguientes secciones:

- **Inicio**
- **¿Quiénes Somos?**
- **Actividades**
- **Novedades**
- **Quiero Ayudar**
- **Contacto**

## Panel de Administración

El panel permite iniciar sesión para habilitar las funciones de alta, edición y eliminación de noticias y actividades.

**Acceso:** `/secciones/formularios/autenticacion/autenticacion.html`

**Usuario:** asociacionsuenios@gmail.com

**Contraseña:** admin

Una vez logueado, se habilitan en las secciones de Novedades y Actividades:
- Botón flotante para **agregar** contenido nuevo.
- Botones de **editar** y **eliminar** en cada tarjeta.
